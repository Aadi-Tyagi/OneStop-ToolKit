Open Command Prompt with admin privileges and copy the following commands one by one.

1.pip -V 
2.pip install tk
3.pip install --upgrade Pillow
4.pip install pefile
5.pip install pyinstaller (Ignore the warning if any while executing this command specifically).
6.pip install auto-py-to-exe 

Note 1: If anyone is facing any errors while following the above command list then consider upgrading your pip by using 
the below mentioned commands:

python -m pip install -upgrade pip

Note 2: Remember after cloning the repo from Github, please download "demo.png" image file in your D drive to run the code without 
hitch or glitch.
